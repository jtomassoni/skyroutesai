# Image Prompts for SkyRoutesAI

This document contains prompts and specifications for generating images for SkyRoutesAI.

## Hero Images

### Main Hero Image
**Prompt:** "Aerial view from airplane window showing clouds and blue sky, soft morning light, travel aesthetic, modern and clean, professional photography style"

**Aspect Ratio:** 3:1 (1200x400px)
**Style Notes:** 
- Soft, calming colors (blues, whites, light grays)
- Airplane window frame visible
- Cloudscape or horizon view
- Bright but not harsh lighting
- Professional travel photography aesthetic

**Usage:** Above-the-fold hero section

---

### Loading State Image
**Prompt:** "Abstract representation of flight paths and routes on a map, minimalist design, soft gradients, blue and white color scheme, modern UI illustration style"

**Aspect Ratio:** 16:9 (1920x1080px)
**Style Notes:**
- Abstract, not literal
- Flight path lines connecting cities
- Soft gradients
- Minimalist design
- Suitable for animation overlay

**Usage:** Loading screen background

---

### Empty State Image
**Prompt:** "Empty airport terminal with soft natural light, minimalist composition, travel aesthetic, calm and inviting atmosphere, professional photography"

**Aspect Ratio:** 4:3 (1200x900px)
**Style Notes:**
- Empty but not desolate
- Clean, modern airport interior
- Soft natural lighting
- Inviting atmosphere
- Professional travel photography

## Style Guidelines

### Color Palette
- Primary: Blue (#2563EB, #3B82F6)
- Secondary: Sky blue (#0EA5E9, #06B6D4)
- Neutral: Grays (#F9FAFB, #F3F4F6, #6B7280)
- Accent: White (#FFFFFF)

### Overall Aesthetic
- **Theme:** Airport lounge / airplane window / travel
- **Mood:** Calm, professional, trustworthy, modern
- **Style:** Clean, minimalist, soft gradients, subtle shadows
- **Photography:** Professional travel photography
- **Illustrations:** Modern, minimalist UI illustrations

### Image Quality Requirements
- High resolution (2x for retina displays)
- Optimized file sizes (WebP format preferred)
- Consistent aspect ratios
- Professional quality
- No watermarks or stock photo credits visible
